Tic Tac Toe

JavaScript – A program implementing the game "tic tac toe". It creates a game board of a specified size and saves how many Xs or Os in a row are needed to win. Players then take turns filling in the grid. The program checks whether the chosen cells are within bounds and ensures that players alternate turns correctly. Once a player reaches the required number of symbols in a row, the program congratulates the winner.
