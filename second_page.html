FIT-Excel

C++ – A semester project in which I created a simplified version of Excel. The program allowed for efficient data storage in a table. If the input was an expression (starting with "="), it was stored in a polymorphic abstract syntax tree (AST) structure, making future evaluations faster than repeatedly calling the parser. The program also allowed the table to be saved in hexadecimal format to a file and subsequently loaded from the file.

Shopping List

C – A C program that takes input for goods on shelves and then shopping lists. It stores the goods in manually managed dynamic arrays and then plans the optimal path through the store to save the customer as much time as possible. The path is then printed as output.

Polymorphic Iterators

PHP – A PHP program that implements an abstract class for individual nodes in a binary tree and also an abstract iterator. The iterators are then specified for three different traversals: Pre-order, In-order, and Post-order, allowing traversal of the tree and output of node values according to these traversal methods.

Custom Priority Queue

JavaScript – A JavaScript program that implements a custom priority queue, allowing the addition of any element to the queue with a numerical priority. A higher priority means the element will take precedence.

Clicker Game

JavaScript – A program implementing a "clicker" game. The user must click as quickly as possible on the displayed balls. After clicking on a ball, the user earns points, and a new ball immediately appears in a different location. If the user fails to click in time, the ball disappears, and a new one is displayed elsewhere.
